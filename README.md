# Python Basics

This repository contains my practice files for learning Python fundamentals step by step.

## Files

- **01_hello.py** 
  first Python script

- **02_data_types.py**
  Introduction to basic data types: integer, float, string and boolean

- **03_variables.py**
  Variable assignment, naming rules, constants and dynamic typing in Python

- **04_input_output.py**
  Basic usage of input() and print() functions

- **05_strings.py**
  Basic string operations: concatenation, f-strings, length and indexing

## Goal
The goal of this repository is to build a foundation in Python programming through small, clear and incremental examples.

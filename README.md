# Python Basics

This repository contains my practice files for learning Python fundamentals step by step.

## Files

- **01_hello.py** 
  first Python script

- **02_data_types.py**
  Introduction to basic data types: integer, float, string and boolean

- **03_variables.py**
  Variable assignment, naming rules, constants and dynamic typing in Python

- **04_input_output.py**
  Basic usage of input() and print() functions

- **05_strings.py**
  Basic string operations: concatenation, f-strings, length and indexing

- **06_data_conversion.py**
  Type conversion : int, float, str, etc

- **07_km_to_miles.py**
  Simple unit conversion program using arithmetic operations and user input

- **08_if_condition.py**
  Basic conditional statements using if, elif, and else

- **09_truthy_falsy.py**
  Exploration of truthy and falsy values in Python and how conditions evaluate them

- **10_comparison_operators.py**
  Practice  with comparison operators(==, !=, >, <, >=, <= ) and boolean results

- **11_logical_operators.py**
  Practice with and, or, not, and boolean expressions

- **12_nested_if.py**
  Rock_Paper_Scissors game using nested if statements

- **13_for_range.py**
  Usage of for loop with range() function

- **14_while_loop.py**
  while loop examples including sum, product, and break behavior

- **15_lists.py**
  List creation, indexing, loops, and built-in list methods

## Goal
The goal of this repository is to build a foundation in Python programming through small, clear and incremental examples.
